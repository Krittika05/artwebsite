# Vincent Van Gogh — A Starry Night Gallery

A stunning tribute gallery celebrating the life, works, and legacy of Vincent van Gogh. Immerse yourself in the swirling skies, golden light, and emotional depth of one of history's greatest artists.

## 🎨 Features

- **Animated Starry Night Background** — A dynamic, interactive sky with twinkling stars, drifting moon, and flowing brushstroke-inspired elements
- **Responsive Masonry Gallery** — 10 of Van Gogh's masterworks displayed in an elegant, responsive layout
- **Interactive Lightbox** — Click any artwork to view full details: title, year, description, medium, location, and dimensions
- **Smooth Scroll Navigation** — Elegant scrolling with fade-in animations and fixed navigation
- **Artistic Typography** — Carefully curated fonts (Playfair Display, Dancing Script, Cormorant Garamond) that evoke the period
- **No Dependencies** — Pure HTML, CSS, and vanilla JavaScript

## 🖼️ Gallery Includes

1. **The Starry Night** (1889) — The iconic swirling sky
2. **Sunflowers** (1888) — Radiant bursts of yellow
3. **The Bedroom** (1888) — Intimate stillness
4. **Irises** (1889) — Garden sanctuary
5. **Café Terrace at Night** (1888) — Gaslit Arles
6. **Wheatfield with Crows** (1890) — Turbulent final vision
7. **Self-Portrait with Bandaged Ear** (1889) — The artist confronts himself
8. **Almond Blossom** (1890) — Hope and new life
9. **The Potato Eaters** (1885) — Raw humanity
10. **Portrait of Dr. Gachet** (1890) — Heartbroken expression

## 🚀 Getting Started

Simply open `index.html` in any modern browser. No build process or server required.

```bash
# Clone or navigate to the project
cd ARTWEBSITE

# Open in browser
open index.html
```
